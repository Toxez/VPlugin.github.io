SpigotMC

Огромный каталог плагинов для серверов. Здесь ты найдешь всё, от простых утилит до сложных систем для управления сервером. Сообщество активно, и многие плагины обновляются регулярно.

SpigotMC (RU)

Русскоязычная версия SpigotMC с плагинами. Удобно, если ищешь что-то на русском. Здесь можно найти переводы популярных плагинов и пообщаться с русскоязычным сообществом разработчиков.

Modrinth

Платформа для модов и плагинов. Поддерживает множество версий Minecraft и предлагает удобный интерфейс для поиска. Также есть система рейтингов и отзывов от пользователей.

Перейти

Fabric

Лучшая платформа для загрузки модов. Fabric позволяет легко устанавливать моды и обеспечивает высокую производительность. Отлично подходит для тех, кто хочет кастомизировать игру.

Перейти

Forge

Популярная платформа для модов Minecraft. Поддерживает огромное количество модов и версий игры. Отличный выбор для тех, кто хочет расширить свой игровой опыт.

Перейти

CurseForge

Платформа для модов, текстур и карт. Имеет удобное приложение для управления модами. Огромное сообщество и множество ресурсов для всех версий Minecraft.

Перейти

PlanetMinecraft

Сообщество для скачивания модов, карт, текстур и скинов. Также есть форум, где можно обсудить проекты и найти идеи для своего сервера.

Перейти

MCreator

Инструмент для создания модов без программирования. Подходит для новичков, которые хотят попробовать себя в моддинге. Поддерживает как Forge, так и Fabric.

Перейти

Bukkit

Классическая платформа для плагинов Minecraft. Хотя сейчас используется реже, всё ещё содержит множество полезных ресурсов для старых версий игры.
